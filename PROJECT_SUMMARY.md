1. Add to `images/` folder:

   - `project1.jpg` to `project4.jpg` - Project screenshots
   - `featured1.jpg` to `featured3.jpg` - Featured project gallery

- Featured: 800×600px or 1200×800px

### Step 3: Test Locally (5 minutes)

1. Open `index.html` directly in browser (double-click the file)
2. Test all interactive features:
   - Dark mode toggle
   - Hamburger menu on mobile
   - Smooth scrolling
   - Form validation
   - Project filtering
   - Testimonial slider

### Step 4: Deploy to Web (10 minutes)

**Option A: GitHub Pages (Recommended - Free)**

1. Create GitHub account at github.com
2. Create new repository named "username.github.io"
3. Upload portfolio folder contents
4. Go to repository Settings → Pages → enable
5. Your site will be live at https://username.github.io

**Option B: Netlify (Easiest - Free)**

1. Go to netlify.com
2. Drag & drop your portfolio folder
3. Instant deployment with generated URL
4. Can add custom domain later

**Option C: Your Own Server**

1. Upload via FTP to your hosting
2. Access via your domain
